# AnkiMarker Config

## Installation

1. Install the [AnkiMarker][anki-marker] add-on.
2. Clone this branch into its `user_files` directory.

    ```shell
    git clone \
        --branch AnkiMarker \
        git@github.com:tnahs/anki-addon-configs.git [PATH-TO-USER-FILES]
    ```

[anki-marker]: https://github.com/tnahs/AnkiMarker
